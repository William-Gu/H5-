HTML5 内容回顾：
 * Canvas（画布）
   * 网页游戏 - 今天完成小游戏
     * H5 Canvas内容
     * javascript内容（面向对象、算法等等）
     * 物理系统（技术模拟现实生活的场景）
     * 精灵系统
     * 动画
   * 页面图表
     * 折线图 - 炒股软件（K线图）
     * 饼状图 - 圆
     * 柱状图 - 矩形
 * 百度地图 - 利用了HTML5 的内容
   * 目前的应用场景非常多
 * Web Socket
   * Socket - 网络协议
     * 多用于游戏中客户端与服务器端的通信
     * 多适用于C/S架构
   * Web Socket是B/S架构下的Socket通信
     * Web Socket API方法或属性即可
 * 拖放API
   * 目前网页中拖放的应用很多 - 并不是HTML5 实现的
   * 一定会被替换成 HTML5 的拖放API
 * 音视频
   * 视频 - 事件、方法和属性
 * Web存储 - Web记事本
   * sessionStorage - 替代Session
   * localStorage - 替代Cookie
 * SVG（矢量图）
   * 与Canvas的区别
 * Web Worker
 * 新表单元素 - 可能成为主流
 * 表单验证 - 并不完善

 * 总结：
   * HTML5 + javascript
